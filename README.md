# Simple WebRTC p2p Video Chat (React)

## Tech

**Client**
create react app (React.js), socket.io-client
**Server**
express, socket.io

## Installation

```sh
$ cd client
$ npm install
$ npm run build
$ cd ..
$ npm install
$ npm run server
```
